# food-divider

Food Divider is an calc app for group meal prep.

## Purpose

Divide the food among the diners, taking into account the weight of the dishes and caloric needs.

## Technologies Stack

backend

- Node.js
- Express.js
- MongoDB

frontend

- React.js

frontend-mobile

- React Native
